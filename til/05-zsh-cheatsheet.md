Some useful shortcuts I use with git plugin ohmyzsh

| Command   | Description                                      |
|-----------|:-------------------------------------------------|
| `gba`     | `git branch -a`                                  |
| `gcam`    | `git commit -am`                                 |
| `gcme`    | `git commit -m`                                  |
| `gco`     | `git checkout`                                   |
| `gd`      | `git fetch --all --prune`                        |
| `gl`      | `git pull`                                       |
| `gp`      | `git push`                                       |
| `gpsup`   | `git push --set-upstream origin [currentbranch]` |
| `gst`     | `git status`                                     |
| `taac`    | `git commit -m`                                  |
